# User Guide / دليل المستخدم

## Installation / التثبيت
### English:
Follow the steps below to install the application:
1. Download the installer from the official website.
2. Run the installer and follow the prompts.
3. Configure your settings as needed.

### العربية:
اتبع الخطوات أدناه لتثبيت التطبيق:
1. قم بتنزيل المثبت من الموقع الرسمي.
2. قم بتشغيل المثبت واتبع التعليمات.
3. قم بتكوين الإعدادات الخاصة بك حسب الحاجة.

## Dashboard Overview / نظرة عامة على لوحة التحكم
### English:
The dashboard provides a comprehensive view of the system's performance. It includes widgets showing key metrics and statistics.

### العربية:
توفر لوحة التحكم رؤية شاملة لأداء النظام. وهذا يتضمن عناصر واجهة المستخدم التي تعرض المقاييس الرئيسية والإحصائيات.

## Security Features / ميزات الأمان

### العربية:
